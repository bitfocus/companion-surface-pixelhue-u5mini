# Connect to PIXELHUE U5 MINI

This document describes how to connect to the PIXELHUE U5 MINI over TCP.

## Fields

You can configure the following fields on **Surfaces** to connect to the U5 MINI:

- **Name**
  - Device name
- **IP Address**
  - Device IP address
- **Port**
  - Default value: `17100`

## Important Notes

The U5 MINI can have only one Companion connection at a time. Connection established later takes precedence and the previous connection is automatically disconnected.
